# TangProject 文档入口

本目录用于帮助新的 Codex 线程快速了解当前项目的代码结构、部署方式和服务器信息。

## 推荐阅读顺序
1. `doc/README.md`
2. `doc/code-structure.md`
3. `doc/server-info.md`

## 目录结构
```text
doc/
  README.md
  code-structure.md
  server-info.md
```

## 文档说明
- `doc/code-structure.md`
  记录当前仓库的代码结构、入口文件、主要接口分组、数据文件和部署脚本分布。
- `doc/server-info.md`
  记录测试服务器与正式服务器的连接信息、部署目录、访问地址、发布入口和常用排查命令。

## 使用建议
- 需要理解项目代码时，先看 `doc/code-structure.md`。
- 需要部署、连服务器、看日志或排查环境问题时，先看 `doc/server-info.md`。
